# Lucky Block — Project Documentation

This repository contains all technical documentation for the Lucky Block project.

## What is Lucky Block?
A mobile puzzle game (iOS & Android) with a Solana blockchain jackpot payout layer.
Built in Godot Engine. Jackpot rewards are bridged on-chain. Gameplay is fully off-chain.

## Repositories

| Repo | Purpose | Visibility |
|------|---------|------------|
| lucky-block-game | Godot mobile game | Private |
| lucky-block-backend | Game server | Private |
| lucky-block-solana-bridge | Jackpot payout service | Private |
| lucky-block-contracts | Solana smart contracts | Public |
| lucky-block-infra | Server infrastructure | Private |
| lucky-block-docs | Project documentation | Public |
